## Spotify Clone

This is a Spotify clone built using React and using JioSaavn API.

<!-- react, tailwindcss -->

## Tech Stack

**Client:** React, TailwindCSS

**Server:** Node, Express

<!--  backend link  -->

## Run Locally

Clone the project

```bash
  git clone https://github.com/kx3ez1/spotify-clone.git
```

Go to the project directory

```bash
  cd spotify-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
